OOP Concept

การเปรียบเทียบในการเขียนโปรเเกรมเเบบ OOP ด้วย ภาษา Java, C++, เเละ Python

การเขียนโปรเเกรมเชิงวัตถุ (Object-oriented Programming หรือ OOP) ด้วยภาษา Java, C++, Python ซึ่งจะมีความเหมือน เเละความเเตกต่างกันหลายๆ ด้าน เช่น การสืบทอดจากหลายคลาส การใช้คลาสเเละออบเจ็กต์ต่างๆ เป็นต้น

หลักการเเละการเปรียบเทียบแนวคิดพื้นฐานของ OOP

1. Class : คือพิมพ์เขียว (blueprint) สำหรับการสร้างวัตถุ เช่น Animal, Dog, Cat ตามตัวอย่างข้างต้น

2. Object : วัตถุหรืออินสแตนซ์คือ สิ่งที่สร้างจากคลาส เช่น a, b, c

3. Subclass/Derived Class : คลาสที่สืบทอดจากคลาสพ่อ/แม่ เช่น Dog, Cat สืบทอดจาก Animal

4. Message : การส่งข้อมูลให้กับวัตถุ เช่น การเรียกใช้ method sound()

5. Inheritance : การสืบทอดคุณสมบัติจากคลาสแม่ (Animal) มายังคลาสลูก (Dog และ Cat)

6. Polymorphism : การใช้ method เดียวกัน (เช่น sound()) แต่ให้ผลลัพธ์ต่างกันตามคลาสที่เป็นวัตถุ

ตัวอย่างเปรียบเทียบสำหรับภาษา Java

class Animal {
    Animal() {
        System.out.println("create animal");
    }
    void sound() {
        System.out.println("animal makes a sound");
    }
}
class Dog extends Animal {
    Dog() {
        System.out.println("dog");
    }
    @Override
    void sound() {
        System.out.println("dog barks");
    }
}
class Cat extends Animal {
    Cat() {
        System.out.println("cat");
    }
    @Override
    void sound() {
        System.out.println("cat meows");
    }
}
public class Main {
    public static void main(String[] args) {
        Animal a = new Dog();
        a.sound(); // Demonstrates polymorphism
    }
}

Example Output:

create animal
dog
dog barks

ตัวอย่างเปรียบเทียบสำหรับภาษา C++

#include <iostream>
using namespace std;

class Animal {
public:
    Animal() {
        cout << "create animal" << endl;
    }
    virtual void sound() {
        cout << "animal makes a sound" << endl;
    }
};

class Dog : public Animal {
public:
    Dog() {
        cout << "dog" << endl;
    }
    void sound() override {
        cout << "dog barks" << endl;
    }
};

class Cat : public Animal {
public:
    Cat() {
        cout << "cat" << endl;
    }
    void sound() override {
        cout << "cat meows" << endl;
    }
};

int main() {
    Animal* a = new Dog();
    a->sound();  // Demonstrates polymorphism
    delete a;
    return 0;
}

Example Output:

create animal
dog
dog barks

ตัวอย่างเปรียบเทียบสำหรับภาษา Python

class Animal:
    def __init__(self):
        print("create animal")
    
    def sound(self):
        print("animal makes a sound")

class Dog(Animal):
    def __init__(self):
        super().__init__()
        print("dog")
    
    def sound(self):
        print("dog barks")

class Cat(Animal):
    def __init__(self):
        super().__init__()
        print("cat")
    
    def sound(self):
        print("cat meows")

a = Dog()
a.sound()  # Demonstrates polymorphism

Example Output:

create animal
dog
dog barks

abstract class คืออะไร

คือ คลาสที่ไม่สามารถสร้างอินสเเตนซ์ได้โดยตรง เเต่จะใช้เป็นเเม่เเบบสำหรับคลาสลูก

ในตัวอย่างนี้ ไม่มีคลาสที่เป็น abstract class เเต่ถ้าต้องการสร้าง abstract class สามารถทำได้โดยใช้ abstract keyword ใน java เเละ C++ หรือ ทำ method เป็น abstract ใน Python
